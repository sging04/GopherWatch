## Launch Codes
```
$ git clone git@github.com:sging04/GopherWatch.git
$ cd GopherWatch
$ pip3 install -r requirements.txt
$ python3 -m venv <name>
$ source <name>/bin/activate
$ cd Go
$ python3 __init__.py
```
